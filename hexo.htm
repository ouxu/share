Hexo

A fast, simple & powerful blog framework

一个快速、简洁且高效的博客框架

Hexo 使用 Markdown 解析文章，在几秒内，即可利用靓丽的主题生成 静态网页。

为什么选择Hexo

  • 简单，傻瓜式操作!
  • 方便，只需要掌握 markdown 语法
  • 快速，生成百篇文章不到一分钟
  • 个性化，可以自定义 Tag 等功能
  • 资源，主题、插件资源丰富
  • 简单，傻瓜式操作！
